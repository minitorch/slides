Sizes

Alternative View

One step -> mat mul

Code

output[b, oc, h, w] = sum([weight[oc, ic, kh, kw] * input[b, ic, h + kh, w + kw]
                           for ic, kh, kw in ...])
# Input image - batch x (in_channel * height * width)
# Weight - (in_channel * kernel_height * kernel_width) x out_channel
